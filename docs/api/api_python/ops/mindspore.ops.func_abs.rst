mindspore.ops.abs
==================

.. py:function:: mindspore.ops.abs(input)

    逐元素计算输入Tensor的绝对值。

    .. math::
        out_i = |input_i|

    参数：
        - **input** (Tensor) - 输入Tensor。其shape为 :math:`(N,*)` ，其中 :math:`*` 表示任意数量的附加维度。

    返回：
        Tensor，shape与 `input` 相同。

    异常：
        - **TypeError** - 如果 `input` 不是Tensor。
