mindspore.ops.Ones
===================

.. py:class:: mindspore.ops.Ones

    创建一个值全为1的Tensor。

    更多参考详见 :func:`mindspore.ops.ones`。

    输入：
        - **shape** (Union[tuple[int], int]) - 指定输出Tensor的shape。
        - **type** (:class:`mindspore.dtype`) - 指定输出Tensor的数据类型。

    输出：
        Tensor，shape和数据类型与输入相同。
