mindspore.ops.square
====================

.. py:function:: mindspore.ops.square(input)

    逐元素返回Tensor的平方。

    .. math::
        y_i = input_i ^ 2

    参数：
        - **input** (Tensor) - 输入Tensor的类型为数值类型。

    返回：
        Tensor，具有与当前Tensor相同的数据类型和shape。

    异常：
        - **TypeError** - `input` 不是Tensor。
