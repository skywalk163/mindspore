mindspore.ops.asinh
====================

.. py:function:: mindspore.ops.asinh(input)

    计算输入元素的反双曲正弦。

    .. math::

        out_i = \sinh^{-1}(input_i)

    参数：
        - **input** (Tensor) - 需要计算反双曲正弦函数的输入。

    返回：
        Tensor，数据类型和shape与 `input` 相同。

    异常：
        - **TypeError** - 如果 `input` 不是Tensor。
