mindspore.ops.Igamma
====================

.. py:class:: mindspore.ops.Igamma

    计算正则化的下层不完全伽马函数。

    .. warning::
        这是一个实验性API，后续可能修改或删除。

    更多参考详见 :func:`mindspore.ops.igamma`。

    输入：
        - **a** (Tensor) - 输入Tensor。
        - **x** (Tensor) - 输入Tensor，数据类型应该与 `a` 一致。

    输出：
        Tensor，数据类型与 `a` 和 `x` 相同。
