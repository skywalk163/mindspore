mindspore.dataset.transforms.Relational
=======================================

.. py:class:: mindspore.dataset.transforms.Relational

    关系运算符。

    可选值如下：

    - ``Relational.EQ`` ：相等。
    - ``Relational.NE`` ：不相等。
    - ``Relational.GT`` ：大于。
    - ``Relational.GE`` ：大于等于。
    - ``Relational.LT`` ：小于。
    - ``Relational.LE`` ：小于等于。
