mindspore.dataset.audio.Interpolation
=====================================

.. py:class:: mindspore.dataset.audio.Interpolation

    音频波形的插值模式。

    可选的枚举值包括： ``Interpolation.LINEAR`` 和 ``Interpolation.QUADRATIC`` 。

    - **Interpolation.LINEAR** - 插值模式为线性。
    - **Interpolation.QUADRATIC** - 插值模式为二次型。
